# ADR-002: Drones para construir geometría de sensing

- **Estado:** Accepted
- **Decisión:** soportar MAVLink/API vendor solo como bridge de telemetría y payload.
- **No decisión:** no implementar autonomía de vuelo dentro de OpenBREC v0.2.
- **Razón:** separación safety-critical, interoperabilidad y adopción por distintos equipos SAR.
