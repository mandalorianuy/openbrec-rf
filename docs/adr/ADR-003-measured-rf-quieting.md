# ADR-003: Aislamiento RF medido, no presumido

- **Estado:** Accepted
- **Decisión:** modelar cortinas/carpas como `IsolationProfile` medido por banda.
- **Razón:** costuras, aperturas, suelo y multipath hacen inválida una cifra nominal de tejido aplicada al conjunto.
- **Safety:** uso temporal, canal de emergencia independiente y prohibición de inferir ausencia de víctima.
